# node-util
Collection of node utilities

## Installation

```
npm i @squirrel-forge/node-util

```

## Usage

```
const util = require( '@squirrel-forge/node-util' );
// util.Class
// util.fn
```

### Classes

 - CliInput
 - Exception
 - FsInterface
 - OutputBuffer

### Functions

 - isPojo
 - isUrl
 - parseInput
 - prompt
 - trimChar
